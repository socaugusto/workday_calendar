# workday_calendar
## Workday calendar toy program

This small library calculates the wordays based on a set of rules such as holidays and working hours.

The library is tested in MSVC 19.44.35207.1 and GCC 13.3.0. It contains a example and unit tests for relevant components.

## How to build

Just open it in VS Code or Visual Studio. Ensure to have a the correct extensions in VS Code.
